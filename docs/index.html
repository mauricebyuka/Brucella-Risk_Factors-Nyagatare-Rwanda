Introduction

This document contains the code used for assessing the risk factors associated with Brucellosis in the Nyagatare district of Rwanda. The results of this analysis have been published in the journal PLOS Neglected Tropical Diseases. This code and associated data are made available to readers who would like to reproduce the results presented in the publication.

To reproduce this analysis, clone (or dowbload) the content of this repository. The R markdown files are in the folder docs. The data are in the "data" folder. The output folder (empty after cloning or downloading will receive files resulting from running the commands. You will ensure that all required libraries are installed prior to running the code chuncs of the R markdown
